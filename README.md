## LFT-Handling Mechanism Project

In dit project ontwikkelen we een automatiseringsoplossing voor TPAC, een innovatief onderzoekscentrum dat zich richt op thermoplastische composieten. Het doel is om het handmatige proces, waarbij gesmolten composietmateriaal van de extruder naar een persvorm wordt overgebracht, te vervangen door een KUKA-robotarm in een ENGEL-insertcel.
